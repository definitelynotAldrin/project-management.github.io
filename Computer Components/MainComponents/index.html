Computer Case

A computer case, is the enclosure that houses and protects the internal components of a computer system. It provides physical support, cooling, and accessibility to the components.

Motherboard

The motherboard is the main circuit board of a computer system that connects and allows communication between various components such as the CPU, memory, and storage devices.

Central Processing Unit

CPU stands for Central Processing Unit, it is the primary component of a computer system responsible for carrying out instructions of a computer program, performing arithmetic and logic operations, and controlling input/output operations.

Cooling Fan

A cooling fan is a component of a computer system that is designed to regulate the temperature of the internal components by moving air across them, helping to prevent overheating and damage.

Graphic Card

A graphic card, also known as (GPU), is a component of a computer system that is designed to handle the rendering images and videos. It is responsible for processing and displaying graphical information, allowing for high-quality and high-speed visual performance.

Optical Disk Drive

An optical disk drive is a component of a computer system that reads and writes data from optical disks, such as CDs, DVDs, and Blu-ray discs. It uses a laser to read and write data on the disk and is typically found in desktop and laptop computers.

Power Supply

A power supply unit (PSU) is a component of a computer system that converts the AC power from a wall outlet into DC power that is usable by the computer's internal components. It regulates the voltage and current supplied to the computer, ensuring that it receives a stable and consistent supply of power.

Random Access Memory

RAM (Random Access Memory) is a type of computer memory that is used to temporarily store data that the CPU (Central Processing Unit) is actively using. It provides faster access to data than other types of storage devices, such as hard disk drives or solid-state drives

Sound Card

A sound card, also known as an audio card, is a component of a computer system that allows the computer to input, process, and output sound.It is responsible for converting analog sound signals to digital signals and vice versa, as well as providing the necessary hardware for audio input and output.
